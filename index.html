About This Project

  • HTML reporting and screenshot capture on failure

Get Started

  1. Clone the repo: git clone https://github.com/rahuljala/AIMCPLAYWRIGT.git
  2. Install dependencies: npm install
  3. Run tests: npx playwright test
  4. View reports: npx playwright show-report
